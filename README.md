# مركز التعليم المستمر - جامعة النهرين

موقع إلكتروني لعرض الدورات التدريبية المتاحة للطلاب.

## المميزات

- عرض الدورات التدريبية المتاحة
- لوحة تحكم للمدير لإضافة دورات جديدة
- تصميم احترافي باللون الأحمر والأبيض والرمادي الفاتح
- واجهة مستخدم كاملة بالعربية مع دعم RTL
- عرض تفاصيل كل دورة بشكل منفصل

## التقنيات المستخدمة

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS

## التثبيت والتشغيل

1. تثبيت المكتبات:
```bash
npm install
```

2. تشغيل المشروع في وضع التطوير:
```bash
npm run dev
```

3. فتح المتصفح على:
```
http://localhost:3000
```

## الصفحات

- `/` - الصفحة الرئيسية مع قائمة جميع الدورات
- `/courses/[id]` - صفحة تفاصيل الدورة
- `/admin` - لوحة التحكم لإضافة دورات جديدة

## البيانات

يتم حفظ الدورات في ملف JSON في مجلد `data/courses.json`. يمكن استبداله بقاعدة بيانات في المستقبل.
