# تعليمات التشغيل

## المتطلبات
- Node.js (الإصدار 18 أو أحدث)
- npm أو yarn

## خطوات التشغيل

1. **تثبيت المكتبات:**
```bash
npm install
```

2. **تشغيل المشروع:**
```bash
npm run dev
```

3. **فتح المتصفح:**
افتح المتصفح على العنوان: http://localhost:3000

## الميزات

### للطلاب (المشاهدين):
- عرض جميع الدورات المتاحة على الصفحة الرئيسية
- النقر على أي دورة لعرض التفاصيل الكاملة
- تصفح معلومات الدورة (السعر، المدة، الموقع، المدرب، إلخ)

### للمدير:
- الانتقال إلى `/admin` للوصول إلى لوحة التحكم
- إضافة دورات جديدة باستخدام النموذج
- عرض جميع الدورات الحالية

## ملاحظات

- يتم حفظ الدورات في ملف `data/courses.json`
- عند أول تشغيل، سيتم إنشاء ملف البيانات تلقائياً مع 6 دورات مثال
- جميع النصوص بالعربية مع دعم RTL (من اليمين لليسار)
- التصميم باستخدام ألوان الأحمر والأبيض والرمادي الفاتح
